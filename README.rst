StratumGS
=========

.. image:: https://badges.gitter.im/stratumgs/stratumgs.svg
   :alt: Join the chat at https://gitter.im/stratumgs/stratumgs
   :target: https://gitter.im/stratumgs/stratumgs?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge

StratumGS is a turn based game engine designed to pit user-designed autonomous
players against each other.

StratumGS is still in a very early stage of development.
